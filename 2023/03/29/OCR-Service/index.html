OCR_Service

Posted on 2023-03-29

前言

本项目使用github上开源模型easyocr进行图像识别。easyocr是基于tesseractOCR，使用python编写的轻量级ocr库，支持八十多种语言的文本识别。

github仓库地址：https://github.com/MichaelKlausL/ocr_service

项目介绍

本次项目使用python编写ocr功能脚本，并将相应依赖一并打包成可执行文件easy_ocr，同学们在使用时只需要下载打包文件，将文件包含到项目中，并调用相应程序即可，不需要额外下载依赖。

项目实现

本栏目介绍项目原理以及python实现方法，具体应用方式可直接跳转至使用方法处。

使用方法

所有语言的使用方法原理一致：

  1. 下载程序包 /easy_ocr

  2. 下载模型，以下提供两种方法

    1. 运行程序包中 /easy_ocr/dist/init.py 文件，这将会为您自动下载预训练好的模型，但可能会耗费一些时间，也有可能连接失败。

    2. 将仓库中的 /.EasyOCR 复制进您电脑中的用户目录中。用户目录地址为：C:\Users\用户名\

    3. 使用相应语言调用程序包中的可执行文件 easyOCR.exe。以下提供 java 、C++和 .Net 的使用方法介绍

程序包介绍

  1. 下载地址 (https://github.com/MichaelKlausL/ocr_service)
  2. 包中共有两个文件夹 /dist 和 /build， 可执行文件位置为 /dist/easyOCR.exe
  3. /dist/easyOCR.exe 接收一个图片的地址，并将识别结果放在当前目录下 ./result/easyocr_text.txt中

命令行

  1. 将当前目录切换到程序包中可执行程序所在位置 cd .../easy_ocr/dist/
  2. 以图片地址为参数调用可执行程序 ./easyOCR.exe ./image01.jpg
  3. 执行完成后，当前目录下会生成 ./result 文件夹，识别结果存储在 ./result/easyocr_text.txt中
